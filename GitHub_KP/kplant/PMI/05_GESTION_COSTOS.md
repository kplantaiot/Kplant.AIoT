## 2. Análisis de Retorno sobre Inversión (ROI)

| Escenario | Fondos Obtenidos | Inversión en Gestión | ROI |
|---|---|---|---|
| **Mínimo** (solo BUILD aprobado) | CLP $15.000.000 (~USD 16K) | CLP $4.000.000 | **275% / 3,75x** |
| **Base** (BUILD + ANID Startup Ciencia) | CLP $149.000.000 (~USD 160K) | CLP $8.000.000 | **1.762% / 18,6x** |
| **Optimista** (BUILD + ANID + IDeA I+D + Semilla Expande) | CLP ~$430.000.000 (~USD 462K) | CLP $12.000.000 | **3.483% / 35,8x** |
| **Máximo** (+ EIC Accelerator) | EUR 2.500.000 (~CLP 3.000.000.000) | CLP $15.000.000 | **19.900% / 200x** |

> **Conclusión:** Incluso en el escenario mínimo, la inversión en gestión de fondos tiene un retorno positivo significativo. El riesgo financiero de postular es muy bajo comparado con el retorno potencial.

## 3. Fuentes de Financiamiento para la Gestión

| Fuente | Monto Disponible | Uso |
|---|---|---|
| Capital propio del equipo fundador | A definir | Fase preparatoria + primeras postulaciones |
| Reinversión de primer fondo obtenido | Variable (% del fondo) | Postulaciones Tier 2 y 3 |
| CORFO Semilla Inicia (si aprobado) | CLP $15.000.000 | Puede financiar dossier, PI y postulaciones siguientes |
| Asesor freelance con fee de éxito | Sin costo fijo | Algunos consultores trabajan con % del fondo aprobado (5–15%) |

## 4. Control de Costos

| Mecanismo | Descripción |
|---|---|
| **Presupuesto por fondo** | Definir techo de gasto antes de iniciar preparación de cada postulación |
| **Hoja de registro de gastos** | Planilla con ítem, proveedor, monto, fecha, fondo asociado |
| **Revisión mensual** | Comparar gasto real vs. presupuestado en reunión de equipo |
| **Alerta de desviación** | Si el costo supera el 20% del presupuesto estimado, revisar prioridades |
| **Documentación contable** | Guardar todas las facturas/boletas para rendiciones futuras de fondos obtenidos |
